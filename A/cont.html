2014 全國資訊能力競賽模擬賽

題目敘述

首先，先恭喜$TFcis$的學弟們，今年又有$5$個人進入全國賽，大家要努力一點，看有沒有人可以直接從全國賽進入選訓營。

接著，進入正題了。還記得從前從前某一次留社考的水題嗎?請參考$Toj16$。但是題目產生了一點微妙的變化，因此接下來就重新敘述一次題目。

今天$S$位同學，每位同學都有自己的心情指數$S_{i}$，但是老師有一個特別的癖好，喜歡一次由兩位同學幫她拿東西，而且同學們只會提跟自己心情指數一樣的重量，所以兩個人可以提起的重量就是$S_{i}+S_{j} (i \neq j)$，假如老師的東西為$W$單位重，那麼只有存在$(i,j)$且$(i \neq j)$使得$S_{i}+S_{j} = W$時老師的東西才能被提起(Hint: $S_{i}$ or $S_{j}$可以為$0$)。不過有一個特例，假如存在$i$使得$2*S_{i} = W$，那麼這個東西也可以被提起。

輸入說明

第一行有一個整數$S$(代表學生人數)，第二行有$S$個正整數$S_{i}$(代表每個學生的心情指數)，第三行有一個整數$Q$(代表老師有$Q$個東西)，接下來有$Q$行，每行有一個正整數$W_{i}$(代表每個東西的重量)。


在所有測試中保證：

  • $0 < S \leq 100000$
  • $0 < S_{i} \leq 100000$
  • $0 < Q \leq 100000$
  • $0 < W_{i} \leq 200000$


輸出說明

對每個$W_{i}$判斷可不可以被提起。若可以請輸出"yes"，否則輸出"no"。

配分方法

  • $10\%~~~~S \leq 10，S_{i} \leq S$
  • $10\%~~~~S \leq 100，S_{i} \leq S$
  • $10\%~~~~S \leq 1000，S_{i} \leq S$
  • $20\%~~~~S \leq 15000，S_{i} \leq S$
  • $50\%~~~~S \leq 100000，S_{i} \leq S$
