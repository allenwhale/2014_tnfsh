D 奇妙的占卜

2014 全國資訊能力競賽模擬賽

題目敘述

$LFsWang$某天心血來潮想要預測一下今年的全國賽誰會脫穎而出，經過$Google$大神的指點，他獲得一項新技能---簡單多邊形占卜術。

顧名思義，簡單多邊形占卜術需要先在紙上畫出一個簡單多邊形，接下來在指上隨便戳一點，如果這個點在這個簡單多邊形內，那就是大吉，如果在邊上，就是普普普普，如果在外面，則是大凶。

但是$LFsWang$不想做這種麻煩事，所以他請你寫個程式幫他占卜。

輸入說明

輸入的第一行有一個整數$N$(代表這個簡單多邊形有$N$個頂點)，接下來的$N$行每行都有兩個整數$x_{i}$,$y_{i}$(代表第$i$個點的座標)，座標按照順時鐘或逆時鐘的順序給出。接下來的有一個整數$Q$(代表有$Q$個人需要占卜),再來的$Q$行每行都有兩個整數$q_{x_{i}}$,$q_{y_{i}}$(代表第i個人代表的座標)。


在所有測試中保證：

  • $2 < N \leq 100$
  • $0 < Q \leq 1000$
  • $|座標| \leq 1000$


輸出說明

輸出每個人的占卜結果，若是大吉，輸出$"good"$,普普輸出$"soso"$,若是大凶則輸出$"bad"$。
